# «Уютный Дом» — Студенческий городок

Информационная система управления студенческим общежитием.

## Описание

Веб-приложение для автоматизации процессов управления студенческим общежитием: учёт студентов, система заявок в службу поддержки, расписание занятий по группам, личные кабинеты для студентов и администраторов.

## Возможности

### Для студентов
- Авторизация по логину и паролю
- Личный кабинет с информацией о студенте (ФИО, группа, корпус, комната)
- Расписание занятий на неделю по группам (101–108)
- Отправка заявок в службу поддержки (сантехника, электрика, мебель, пропуск, уборка, другое)
- Просмотр своих заявок с ответами администрации и датой визита мастера
- Скрытие прочитанных заявок
- Сброс забытого пароля
- Тёмная / светлая тема

### Для администраторов
- Панель управления студентами (добавление, редактирование, удаление)
- Массовое добавление студентов по группам (по 10 человек)
- Управление заявками: ответы, назначение мастера, смена статуса
- Скрытие / отображение заявок
- Поиск студентов по ФИО и фильтрация по группам
- Статистика: студенты по группам, заявки по статусу, по типу, по месяцам
- Уведомления о новых заявках в реальном времени (Socket.IO)

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Сервер | Node.js, Express.js |
| База данных | SQLite3 |
| Фронтенд | HTML5, CSS3, JavaScript |
| Аутентификация | express-session, bcrypt |
| Real-time | Socket.IO |
| Стили | Glassmorphism, адаптивный дизайн, тёмная тема |
| Безопасность | helmet, express-rate-limit, параметризованные SQL |

## Структура базы данных

### Таблица `students_groups`
| Поле | Тип | Описание |
|------|------|---------|
| id | INTEGER | ID студента (автоинкремент) |
| login | TEXT | Логин (уникальный) |
| password | TEXT | Хеш пароля (bcrypt) |
| full_name | TEXT | ФИО студента |
| subgroup | TEXT | Подгруппа |
| group_number | TEXT | Учебная группа (101–108) |
| room | TEXT | Номер комнаты |
| email | TEXT | Email |
| phone | TEXT | Телефон |
| created_at | TEXT | Дата создания |
| updated_at | TEXT | Дата обновления |

### Таблица `requests`
| Поле | Тип | Описание |
|------|------|---------|
| id | INTEGER | ID заявки (автоинкремент) |
| student_id | TEXT | Логин студента |
| type | TEXT | Категория заявки |
| message | TEXT | Описание проблемы |
| date | TEXT | Дата создания |
| admin_reply | TEXT | Ответ администратора |
| master_datetime | TEXT | Дата и время визита мастера |
| status | TEXT | open / in_progress / done |
| hidden_by_student | INTEGER | Скрыто студентом (0/1) |
| hidden_by_admin | INTEGER | Скрыто администратором (0/1) |

## Установка и запуск

```bash
# Клонировать репозиторий
git clone <url>
cd dormitory-project

# Установить зависимости
npm install

# Настроить переменные окружения (.env)
# PORT=3001
# ADMIN_LOGIN=admin
# ADMIN_PASSWORD=12345
# SESSION_SECRET=your-secret-key

# Запустить сервер
node server.js
```

Сервер будет доступен по адресу: **http://localhost:3001**
